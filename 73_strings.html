Stanford Sentiment Treebank V1.0

This is the dataset of the paper:

Recursive Deep Models for Semantic Compositionality Over a Sentiment Treebank Richard Socher, Alex Perelygin, Jean Wu, Jason Chuang, Christopher Manning, Andrew Ng and Christopher Potts Conference on Empirical Methods in Natural Language Processing (EMNLP 2013)

If you use this dataset in your research, please cite the above paper.

@incollection{SocherEtAl2013:RNTN, title = {{Parsing With Compositional Vector Grammars}}, author = {Richard Socher and Alex Perelygin and Jean Wu and Jason Chuang and Christopher Manning and Andrew Ng and Christopher Potts}, booktitle = {{EMNLP}}, year = {2013} }

This file includes:

  1. original_rt_snippets.txt contains 10,605 processed snippets from the original pool of Rotten Tomatoes HTML files. Please note that some snippet may contain multiple sentences.

  2. dictionary.txt contains all phrases and their IDs, separated by a vertical line |

  3. sentiment_labels.txt contains all phrase ids and the corresponding sentiment labels, separated by a vertical line. Note that you can recover the 5 classes by mapping the positivity probability using the following cut-offs: [0, 0.2], (0.2, 0.4], (0.4, 0.6], (0.6, 0.8], (0.8, 1.0] for very negative, negative, neutral, positive, very positive, respectively. Please note that phrase ids and sentence ids are not the same.

  4. SOStr.txt and STree.txt encode the structure of the parse trees. STree encodes the trees in a parent pointer format. Each line corresponds to each sentence in the datasetSentences.txt file. The Matlab code of this paper will show you how to read this format if you are not familiar with it.

  5. datasetSentences.txt contains the sentence index, followed by the sentence string separated by a tab. These are the sentences of the train/dev/test sets.

  6. datasetSplit.txt contains the sentence index (corresponding to the index in datasetSentences.txt file) followed by the set label separated by a comma: 1 = train 2 = test 3 = dev

Please note that the datasetSentences.txt file has more sentences/lines than the original_rt_snippet.txt. Each row in the latter represents a snippet as shown on RT, whereas the former is each sub sentence as determined by the Stanford parser.

For comparing research and training models, please use the provided train/dev/test splits.

def process_sentences(BASE_PATH):

  # define each individual files path by appending their names
  datasetSenteces_path = BASE_PATH + '/datasetSentences.txt'
  datasetSplit_path = BASE_PATH + '/datasetSplit.txt'
  dictionary_path = BASE_PATH + '/dictionary.txt'
  sentiment_labels_path = BASE_PATH + '/sentiment_labels.txt'

  # read each file by either read_csv or read_table and add header=None if there is no header and add separator 
  df_sentences = pd.read_table(datasetSenteces_path)
  df_split_label = pd.read_csv(datasetSplit_path)
  df_dictionary =  pd.read_table(dictionary_path, sep='|', header=None)
  df_sentiment_label = pd.read_table(sentiment_labels_path, sep='|')

  # printing shape of each one
  print(df_sentences.shape)
  print(df_split_label.shape)
  print(df_dictionary.shape)
  print(df_sentiment_label.shape)
 
  # sort by sentence_index because sentences and split_label have same shape with one to one mapping
  df_sentences.sort_values('sentence_index')
  df_split_label.sort_values('sentence_index')

  # combine them to get processed sentences dataframe 
  # df_sentences_processed = pd.concat([df_sentences, df_split_label], axis=1)
  # using inner join 
  df_sentences_processed = pd.merge(left=df_sentences, right=df_split_label, left_on='sentence_index', right_on='sentence_index')
  
  # sentence_index is no more required -> drop it from dataframe
  df_sentences_processed.drop('sentence_index', axis='columns', inplace=True)
  print(df_sentences_processed.head(3))

  # for dictionary and sentiment_label define short name for their columns
  df_dictionary.columns = ['phrase','phrase_id']
  df_sentiment_label.columns = ['phrase_id','sentiment_value']  

  # sort by phrase_id because dictonary and sentiment_value for that have one to one mapping with having same shape
  df_dictionary.sort_values('phrase_id')
  df_sentiment_label.sort_values('phrase_id')

  # combine them so we can get dictionary_processed 
  # df_dictionary_processed = pd.concat([df_dictionary, df_sentiment_label], axis=1)
  # using inner join 
  df_dictionary_processed = pd.merge(left=df_dictionary, right=df_sentiment_label, left_on='phrase_id', right_on='phrase_id')

  # phrase_id is no more required -> drop it.
  df_dictionary_processed.drop('phrase_id', axis='columns', inplace=True)
  print(df_dictionary_processed.head(3))

  # apply inner join on sentence and phrase column from sentences-processed and dictionary_processed respectively.
  df_final = pd.merge(left=df_sentences_processed, right=df_dictionary_processed, left_on='sentence', right_on='phrase')
  
  df_final.drop('phrase',axis='columns',inplace=True)
  print(df_final.shape)
  print(df_final.head(5))
  print(df_final.columns)

  return df_final

def get_frequent_ngram(tokens, ngram, n=20):
    """Find most common n n-grams in tokens."""
    n_grams = ngrams(tokens, ngram)
    freq_dist = FreqDist(n_grams)
    top_freq = freq_dist.most_common(n)
    return pd.DataFrame(top_freq, columns=["ngram", "count"])

  sentece = deEmojify(sentence)

  data_index_np = np.array(data_index)

  padded_array = np.zeros(56)
  padded_array[:data_index_np.shape[0]] = data_index_np
  data_index_np_pad = padded_array.astype(int)
  live_list.append(data_index_np_pad)
  live_list_np = np.asarray(live_list)
  
  score = trained_model.predict(live_list_np, batch_size=1, verbose=0)
  # print(score)

  return score

def test_api(BASE_DIR, data):
  word_idx, weight_matrix, trained_model = init_api(BASE_DIR)

  predictions = []
  i = 1
  for each in data['sentence']:
    prediction = predict_senti_api(str(each), word_idx, weight_matrix, trained_model)
    idx = np.argmax(prediction)
    categories = ["very negative", "negative", "neutral", "positive", "very positive"]
    predictions.append(categories[idx])
    # print("review number - ", i, categories[idx])
    i += 1

  return predictions
